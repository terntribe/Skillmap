## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL 14+
- Docker (optional)

### Installation

1. Clone the repo:

   ```bash
   git clone https://github.com/Incognitol07/Skillmap.git
   cd Skillmap
   ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Setup environment:

    ```bash
    cp .env.example .env
    ```

### Edit .env with your credentials

Start PostgreSQL:

```bash
docker-compose up -d
```

Run the app:

```bash
npm run dev
```
